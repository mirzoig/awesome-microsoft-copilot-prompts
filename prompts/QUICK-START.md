## üìù Writing & Communication (10 prompts)

### 1. Professional Email Draft

**Use Case:** Write business emails quickly with the right tone

**Prompt:**
```
Draft a professional email to [recipient] about [topic]. 
The tone should be [formal/friendly/urgent]. 
Include: [key points you want to make].
Keep it under 150 words.
```

**Example:**
```
Draft a professional email to my team about the project deadline extension. 
The tone should be friendly but clear. 
Include: new deadline is March 15, reasons for extension, what hasn't changed.
Keep it under 150 words.
```

---

### 2. Difficult Conversation Script

**Use Case:** Prepare for tough conversations

**Prompt:**
```
I need to [deliver bad news/decline a request/address poor performance] with [person/team]. 
Help me structure this conversation:
1) Opening statement
2) Main message
3) Reasoning
4) Next steps
5) Closing on a positive note
```

---

### 3. Meeting Request Email

**Use Case:** Schedule meetings professionally

**Prompt:**
```
Draft a meeting request email for [topic] with [attendees]. 
Meeting details: [date/time/duration].
Agenda should cover: [key topics].
Make it clear why their attendance matters.
```

---

### 4. Thank You Note

**Use Case:** Professional gratitude messages

**Prompt:**
```
Write a professional thank you note to [person] for [reason]. 
Mention specifically: [what they did that helped].
Tone should be genuine and warm, not overly formal.
Length: 3-4 sentences.
```

---

### 5. Apology Message

**Use Case:** Own mistakes professionally

**Prompt:**
```
I need to apologize for [situation] to [person/team].
Help me write a message that:
1) Takes clear responsibility (no excuses)
2) Explains what happened
3) Shows how I'll prevent it again
4) Requests a chance to make it right
```

---

### 6. LinkedIn Connection Message

**Use Case:** Network effectively

**Prompt:**
```
Write a LinkedIn connection request to [person's role] at [company].
Context: [how you found them / mutual connection / why connecting].
Make it personalized and professional. 2-3 sentences max.
```

---

### 7. Rejection Email (Kind)

**Use Case:** Decline offers or requests gracefully

**Prompt:**
```
I need to decline [request/offer] from [person] because [reason].
Write a message that:
1) Thanks them for the opportunity
2) Declines clearly (not "maybe later")
3) Explains briefly if appropriate
4) Keeps the relationship positive
```

---

### 8. Follow-Up Message

**Use Case:** Professional persistence without being annoying

**Prompt:**
```
I sent [type of message] to [person] about [topic] on [date] and haven't heard back.
Write a follow-up that:
1) References the original message
2) Adds new value or context
3) Includes clear call-to-action
4) Doesn't sound desperate or pushy
```

---

### 9. Announcement Draft

**Use Case:** Share news with your team/organization

**Prompt:**
```
Draft an announcement about [news/change] for [audience].
Structure:
1) What's happening
2) Why it matters
3) What changes (if anything)
4) Next steps
5) Who to contact with questions

Tone: [professional/enthusiastic/serious]
```

---

### 10. Constructive Feedback

**Use Case:** Give feedback that actually helps

**Prompt:**
```
I need to give feedback to [person] about [situation/behavior].
Frame it constructively:
1) Specific observation (not judgment)
2) Impact of the behavior
3) Desired outcome
4) Offer to support improvement

Keep it focused on growth, not criticism.
```

## üéì Learning & Research (10 prompts)

### 1. Explain Complex Topic Simply

**Use Case:** Understand difficult concepts

**Prompt:**
```
Explain [complex topic] to me like I'm [age/profession with no background in this].
Use analogies and real-world examples.
Break it into 3-5 key concepts.
No jargon unless you explain it first.
```

1) Pros and cons of each
2) Cost implications
3) Time investment
4) Best use cases for each
5) Your recommendation with reasoning
```

## üìä Analysis & Problem-Solving (10 prompts)

### 1. Data Analysis (Pasted Content)

**Use Case:** Make sense of numbers

**Prompt:**
```
Analyze this data: [paste data/table]
Provide:
1) Key trends or patterns
2) Outliers or anomalies
3) What this tells us
4) Potential issues or red flags
5) Recommended actions

---

### 2. Root Cause Analysis

**Use Case:** Get to the real problem

**Prompt:**
```
Problem: [describe issue]
Help me find the root cause:
1) Surface symptoms (what we can see)
2) Ask "why" 5 times to dig deeper
3) Potential root causes ranked by likelihood
4) How to verify which is the actual root cause
5) Solutions that address the root, not symptoms
```

---

### 3. Pros & Cons List

**Use Case:** See both sides clearly

**Prompt:**
```
I'm considering: [decision/action]
Create a comprehensive pros & cons analysis:
1) Short-term pros & cons
2) Long-term pros & cons
3) Risks vs. opportunities
4) What we might be overlooking
5) Questions to ask before deciding
Be realistic, not optimistic or pessimistic.
```

---

### 4. Problem Reframing

**Use Case:** Find better solutions by asking better questions

**Prompt:**
```
Current problem statement: [how I'm thinking about this]
Help me reframe it:
1) Is this the real problem or a symptom?
2) Who else is affected (stakeholders)?
3) What if we're solving the wrong problem?
4) Alternative ways to frame this challenge
5) Which framing opens up the most solutions?
```

---

### 5. Risk Assessment

**Use Case:** Identify what could go wrong

**Prompt:**
```
I'm planning to: [action/project]
Assess risks:
1) What could go wrong (brainstorm everything)
2) Likelihood of each (high/medium/low)
3) Impact if it happens (high/medium/low)
4) Top 3 risks to mitigate
5) Mitigation strategies for each
6) Early warning signs to watch for
```

---

**Use Case:** Evaluate potential moves

**Prompt:**

Opportunity: [describe situation/offer]

1) What's the actual opportunity here?

## üéØ Quick Tips for Better Prompts

### 1. Be Specific
‚ùå "Write an email"  
‚úÖ "Draft a professional email to my team about the Q2 deadline extension, friendly tone, under 150 words"

### 2. Provide Context
Tell Copilot:
- Who the audience is
- What you're trying to achieve
- Any constraints (time, length, tone)

### 3. Use Examples
"Like this: [example]" helps Copilot understand what you want

### 4. Iterate
If the first result isn't perfect:
- "Make it more concise"
- "Add more specific examples"
- "Change the tone to be more formal"

### 5. Save What Works
Build your own prompt library from ones that give great results
